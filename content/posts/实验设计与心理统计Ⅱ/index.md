## 第8题

医学上测定，正常人的血色素浓度是每100毫升13克，某学校进行抽查，37名学生的血色素平均值 $\bar{X} = 12.1$（克/100毫升），标准差 $s = 1.5$（克/100毫升），试问该校学生的血色素是否显著低于正常值？

**解答**：

*  * 原假设 $H_0: \mu = 13$（该校学生血色素不低于正常值）  
    * 备择假设 $H_1: \mu < 13$（显著低于正常值，单侧左尾检验）

*  由于总体方差未知，使用单样本 $t$ 检验：
    $$t = \frac{\bar{X} - \mu_0}{s / \sqrt{n}} = \frac{12.1 - 13}{1.5 / \sqrt{37}}$$
    
    计算标准误：
    $$s / \sqrt{n} = 1.5 / \sqrt{37} \approx 1.5 / 6.0828 \approx 0.2466$$
    
    得：
    $$t = \frac{-0.9}{0.2466} \approx -3.65$$
    自由度 $df = n - 1 = 36$。

*  取显著性水平 $\alpha = 0.05$，查 $t$ 分布表得单侧临界值 $t_{0.05}(36) \approx 1.69$，拒绝域为 $t < -1.69$。  
    计算得 $t = -3.65 < -1.69$，落入拒绝域，故拒绝原假设。

*  该校学生的血色素显著低于正常值（$p < 0.05$）。

## 第9题

12名被试作为实验组，经过训练后测量深度知觉，结果误差的平均 $\bar{X}_1 = 4$ cm，标准差 $s_1 = 2$ cm；另外12名被试作为控制组不参加任何训练，测量结果 $\bar{X}_2 = 6.5$ cm，$s_2 = 2.5$ cm，问训练是否明显减小了深度知觉的误差？

**解答**：

*  * 原假设 $H_0: \mu_1 = \mu_2$（训练无效果，误差无差异）  
    * 备择假设 $H_1: \mu_1 < \mu_2$（训练减小误差，单侧左尾检验）

*  采用两样本 $t$ 检验：
    
    * 合并方差：
        $$s_p^2 = \frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1 + n_2 - 2} = \frac{11 \times 2^2 + 11 \times 2.5^2}{22} = \frac{44 + 68.75}{22} = 5.125$$
        $$s_p = \sqrt{5.125} \approx 2.264$$
        
    * 标准误：
        $$SE = s_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}} = 2.264 \times \sqrt{\frac{1}{12} + \frac{1}{12}} \approx 2.264 \times 0.4082 = 0.924$$
        
    * 检验统计量：
        $$t = \frac{\bar{X}_1 - \bar{X}_2}{SE} = \frac{4 - 6.5}{0.924} = \frac{-2.5}{0.924} \approx -2.705$$
        自由度 $df = n_1 + n_2 - 2 = 22$。

  取 $\alpha = 0.05$，单侧临界值 $t_{0.05}(22) = 1.717$，拒绝域为 $t < -1.717$。  
    计算得 $t = -2.705 < -1.717$，拒绝原假设。

  训练显著减小了深度知觉误差（$p < 0.05$）。
